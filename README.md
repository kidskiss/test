## 转义
\\ 反斜杠 
\` 反引号 
\* 星号 
\_ 下划线 
\{} 大括号 
\[] 中括号 
\() 小括号 
\# 井号 
\+ 加号 
\- 减号 
\. 英文句号 
\! 感叹号

## 快捷键
ctrl+1 一级标题<br> 
ctrl+2 二级标题 
····
ctrl+shift+o 有序列表
ctrl+u 无序列表
ctrl+g 插入图片
ctrl+l 插入超链接
Ctrl+B　粗体 
Ctrl+I　斜体
Ctrl+Q　引用 
Ctrl+K　代码块
